Installation
============

Debian/Ubuntu
-------------

.. code:: bash

    wget -q https://raw.github.com/brbsix/vagrant-bash-completion/master/etc/bash_completion.d/vagrant
    sudo install -m 0644 vagrant /etc/bash_completion.d/vagrant-test
